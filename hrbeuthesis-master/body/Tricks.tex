\BiSection{引言}{Something About Templates}
\label{tricks:abouttemplates}
本章面向其水下抓取作业需求，研制一款水下绳驱机械臂。该机械臂采用绳驱动结构，将质量较大的驱动电机后置于基座，从而有效减小运动过程中对运载器的扰动。为此，本章围绕绳驱机械臂的结构设计展开研究，具体内容包括：总体方案设计、本体结构设计、驱动绳走线分析、驱动密封舱设计。
\BiSection{水下绳驱机械臂设计要求}{Something About Templates}
针对搭载于微小型运载器的对海底作业的绳驱机械臂，在设计时需要达到的以下四个设计要求：

(1) 机械臂安装于运载器的正下方，能够完全折叠；

(2) 机械臂完全伸展下探深度要大于500mm；

(3) 机械臂运动部分重量不大于3Kg；

(4) 机械臂承载能力1Kg的有效负载。
\BiSection{机械臂总体结构设计}{Something About Templates}

(1)关节1水平旋转关节设计

水平旋转关节采用基于欧拉绞盘原理的单绳循环驱动方案。该机构利用绳索与绕线盘之间的摩擦力实现力矩传递，具体设计如下：
\begin{itemize}
    \item \textbf{驱动原理与缠绕逻辑}：该关节由单根高强度驱动绳构成闭环传动回路。如图\ref{pic14}所示，驱动绳起始于驱动绞盘（主动轮），经特定路径绕至关节旋转绞盘（从动轮），最终返回并固定于同一驱动绞盘上。驱动绳在驱动轴上采取“一端正绕、一端反绕”的配置，当电机带动驱动轴旋转时，绳索的一端被收卷，另一端被释放，从而驱动关节绞盘同步转动。

    %添加图片；[]中为可选参数，可以设置图片的宽高；{}中为图片的相对位置
    \includegraphics[width = 0.7\textwidth]{f1}
    \caption{水平旋转关节原理} % 图片标题

    \item \textbf{绞盘放大效应与防滑设计}：为确保动力传递的可靠性并防止非线性打滑，本设计利用了绞盘方程（Capstan Equation）：
    \begin{equation}
        T_{load} = T_{hold} \cdot e^{\mu\phi}
    \end{equation}
    其中，$T_{load}$ 为负载端张力，$T_{hold}$ 为紧边张力，$\mu$ 为绳索与盘面的摩擦系数，$\phi$ 为绳索在绞盘上的总包角。通过增加驱动绳在绞盘上的缠绕圈数（即增大包角 $\phi$），可以在较低的初始张力下获得较大的摩擦力，有效避免了动态负载下的绳索打滑现象。
\end{itemize}

(2)关节2肘部双铰链滚动关节

为兼顾驱动力矩输出与收拢空间限制，肘部关节采用了增力机构与纯滚动接触面相结合的设计方案。
\begin{itemize}
\item \textbf{增力传动机构}

如图\ref{fig:total_figure2}所示，肘部关节集成了一套滑轮组倍率放大机构，通过物理结构的增益设计实现了驱动力矩的显著提升：

\begin{itemize}
\item \textbf{滑轮组倍率放大}：两根驱动绳索在定滑轮与动滑轮之间进行多次往复缠绕。根据滑轮组机械增益原理，该结构将电机提供的初始张力按绕组倍数放大，从而在不增加电机功率的前提下，极大地提升了肘部关节的末端负载能力，有效适应水下复杂作业环境对大推力的需求。
\item \textbf{单轴差动缠绕布局}：两根驱动绳索的末端分别以正向和反向缠绕的形式固定于同一电机驱动轴上。这种设计确保了电机旋转时，一侧绳索收卷的同时另一侧等量释放。该配置不仅维持了系统在全行程中的张力恒定，消除了绳索松弛风险，还通过共轴驱动实现了结构的极度紧凑化。
\end{itemize}

\BiSubsection{驱动绳走线及导向设计}{Which Templates I should Select？}


在绳驱机械臂中，绳索作为动力与运动的载体，其路径规划与导向方式直接影响系统的传动精度、能量效率及使用寿命。

目前，常用的驱动绳布置方式主要有两类。第一类是采用硬质导管，将驱动绳索直接引导至目标关节处。该方法操作相对简便，只需将按特定弯曲方式加工好的导管固定在机械臂上即可。然而，这种方式存在明显缺陷：导管与驱动绳索之间的滑动摩擦，不仅会造成驱动力损耗，还会显著缩短绳索的使用寿命。
第二类为滑轮引导方式，即通过安装滑轮，将绳索引导至预定位置。该方法的安装过程较为复杂，且随着机械臂自由度的增加，所需滑轮轮的数量增多，导向难度也随之上升。
但是，滑轮将绳索与固定结构之间的滑动摩擦转化为滚动摩擦，显著降低了摩擦损耗与绳索磨损，是目前应用最广泛、技术最成熟的导向方案。
综合对比两种布置方式，并结合本文设计需求，最终选用滑轮导向的方法。

(1)滑轮导向的基本原理

滑轮导向的核心思想是：在绳索路径的每一个转折点或支撑点处，安装一个可绕自身轴线自由旋转的滑轮，绳索绕过滑轮绳槽，通过滑轮与轴之间的滚动轴承实现低摩擦转向。

具体特性如图\ref{pic23}：

\begin{itemize}
\item {力学特性:}
当绳索绕过理想滑轮（不计轴承摩擦）时，滑轮两侧绳索的张力 $T_1$ 与 $T_2$ 满足：
\begin{equation}
      T_1 = T_2
\end{equation}
即滑轮仅改变力的方向，而不改变力的大小。实际中由于轴承摩擦及绳索与绳槽的微小滑动，存在效率系数 $\eta < 1$，但通过选用低摩擦轴承（如深沟球轴承、陶瓷轴承）可使得 $\eta > 0.95$。

\item{运动特性:}
绳索与滑轮之间满足无滑动的滚动关系，绳索线速度 $v$ 与滑轮角速度 $\omega$、滑轮半径 $R$ 满足：
\begin{equation}
      v = \omega R
\end{equation}
因此，通过检测滑轮转角可间接推算绳索位移，为关节位置反馈提供一种冗余测量手段。
\end{itemize}

\BiSubsection{驱动系统密封舱设计}{Which Templates I should Select？}

驱动密封舱作为水下机械臂的物理载荷中心，其性能直接影响整机的可靠性。在设计过程中，主要考虑了以下几个核心功能：
\begin{itemize}
    \item \textbf{环境防护与承压}：确保内部电子元器件、驱动及传动组件与高压海水完全隔离，防止电气故障；
    \item \textbf{水动力性能优化}：通过流线型构造降低运动阻力，提升机械臂在水下的能源效率；
    \item \textbf{结构支撑}：为复杂的组件布局提供稳固的安装基准，满足机械臂灵活运动的力学需求。
\end{itemize}

由于密封舱内集成了控制单元与驱动电机，控制元件需固定于舱体内壁，而电机则需要通过输出轴将转矩传递至舱外，因此该密封舱涉及了静密封与动密封的组合设计。舱内安装的具体组件参数如表 \ref{tab:cabin_components} 所示。

\BiSection{水下绳驱机械臂样机集成}{Something About Templates}
在完成本体结构设计与驱动密封舱研制后，本节开展了水下绳驱机械臂的样机集成与系统调试工作。集成过程主要分为硬件装配、绳索张紧调校及控制系统联调三个阶段。

首先，按照“先内部、后外部”的原则进行装配。在驱动密封舱内部，完成 STM32 控制核心、驱动器及电压转换模块的紧凑堆叠与接线；随后，通过分体式动密封组件将三组 Faulhaber 电机输出轴引出舱外，并安装驱动绞盘。为确保密封可靠性，所有静密封界面均均匀涂抹真空硅脂，并利用压力测试系统验证了密封舱在 0.5MPa 环境下的气密性。

其次，进行关键的绳索路径布设。驱动绳索通过水平旋转关节内部的过绳通道，经由各级导向滑轮组引导至肘部与腕部关节。利用本章设计的同轴反向单向轴承张紧机构，对每组驱动回路进行了初始预紧力调节。通过这种方式，有效消除了绳索在传动过程中的非线性迟滞，确保了驱动力矩的线性传递。

最后，进行了系统级联调。通过上位机发送控制指令，验证了机械臂从完全折叠姿态到最大下探深度姿态（730mm）的平稳切换。实测样机总质量为 1.8kg，完全符合设计指标中对于微小型运载器载荷限制的要求。集成完成后的水下绳驱机械臂样机实物如图 \ref{fig:total_figure18} 所示。

\BiSection{本章小结}{Something About Templates}
本章针对微小型水下运载器对作业机械臂轻量化、紧凑化及低扰动的迫切需求，完成了一款三自由度水下绳驱机械臂的研制工作，主要研究成果如下：

\begin{enumerate}
    \item \textbf{总体与本体设计}：提出了一种可完全折叠的绳驱构型方案，通过关联双铰链滚动机构解决了常规关节收拢空间受限的问题，实现了机械臂在运载器底部的紧凑嵌套。
    \item \textbf{传动与导向优化}：针对绳驱系统的摩擦损耗问题，设计了基于全滑轮导向的走线方案；并创新性地利用单轴差动缠绕布局与滑轮组增力机构，在不增加电机功率的前提下显著提升了关节的负载能力。
    \item \textbf{可靠性密封研制}：研制了高度集成的驱动密封舱，通过分体式动密封结构解决了小尺寸格莱圈的安装难题。实测样机在保证 730mm 下探深度的同时，水中重量仅为 0.4kg，各项指标均满足预设的技术要求。
\end{enumerate}

本章研究成果为后续开展水下机械臂的运动学建模及精细化控制奠定了坚实的硬件基础。
