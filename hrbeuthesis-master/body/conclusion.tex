\BiAppendixChapter{结~~~~论}{Conclusions}



本文面向水下微小型运载器的空间受限与精细作业痛点，摒弃了传统机械臂自重偏大、
水流扰动剧烈的刚性驱动路线，尝试将绳索传动机制引入水下机械臂的开发中。各项工作严格按照工程逻辑推进，具体总结如下：

\begin{enumerate}
    \item \textbf{提出并实现了一种高负载自重比的水下绳驱机械臂总体结构方案}
    研制了具备高负载自重比的绳驱机械臂物理样机。在构型布局上，
    将电机及主控板统一剥离至基座后端的独立密封舱内，
    减小了运动部件质量。针对运载器底盘腹部的狭小安装空间，
    开发了纯滚动双铰链机构以实现整臂的极限折叠...
    设计了滑轮组机构与单轴差动缠绕导向方案，
    大幅提升了末端承载能力。
    通过理论力学计算与 ANSYS 有限元分析，
    验证了核心滑轮轴、连接件及电机驱动轴在极限工况下的结构强度满足水下作业的安全需求。
    
    \item \textbf{梳理了耦合约束下的运动学机理与轨迹平滑策略}
    运用改进 D-H法推导正解方程，并借助数值迭代和映射矩阵化解了绳驱
    特有的多关节联动逆解难题。
    为了机械臂工作平稳，
    对梯形速度规划与五次多项式算法的插补效果进行了横向对比
    并在关节空间与笛卡尔空间分别实现了平滑的直线及圆弧运动轨迹生成。
    MATLAB 仿真结果表明，所提算法有效抑制了速度和加速度突变，
    显著减小了机械臂运动对水体的冲击扰动。
    
    \item \textbf{构建了基于单目视觉的目标定位与自主抓取策略}
    建立了水下环境的单目相机模型，
    基于 ArUco 标签和“眼在手外”构型，
    通过 Tsai-Lenz 两步法完成了高精度的手眼标定。
    打通了图像像素空间到机械臂物理空间的坐标转换链路，
    并设计了基于 ROS 2 架构的抓取任务节点，
    实现了对目标物体的动态定位与抓取动作的自主规划。
    
    \item \textbf{实施了整机软硬件联调与闭环测试}
    构建了“树莓派+STM32”的主从式计算架构，
    并在底层部署了带抗干扰校验的通信架构。
    实测数据证实，样机最大工作半径达 730mm 。
    在视觉抓取及定轨运行实验中
    机械臂具备良好的运动响应性能与定位精度，
    视觉引导下的闭环抓取动作连贯且成功率高，
    充分验证了整套软硬件系统的可行性与理论模型的正确性。
\end{enumerate}


尽管本文在水下绳驱机械臂的设计与控制上取得了一定成果，
但仍存在一些需要进一步深化的研究方向：

\begin{enumerate}
    \item \textbf{绳索柔性动力学与误差补偿}：
    本文运动学模型基于刚性假设，
    未充分考虑高负载与长时运行下绳索的形变。
    未来可引入绳索柔性动力学模型与张力反馈闭环机制，
    以进一步提升末端的绝对定位精度。
    
    \item \textbf{水动力学建模与抗扰动控制}：
    水下流阻及附加质量力不可忽略。
    未来的底层算法需结合 CFD 仿真参数，开发自适应控制算法，
    以抵抗复杂洋流带来的非线性干扰。
    
    \item \textbf{深水高压环境的真实海试验证}：
    目前样机试验主要在实验室陆地环境中完成，尚未进行水下测试。
\end{enumerate}
